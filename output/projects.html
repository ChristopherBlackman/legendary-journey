Mid-Point Displacement 2D

Just me playing with fractals and procedural world generation. I found a neat web site that explained the mid-point displacement algorithm and after a long walk I decided that I wanted to try making it in python with pygame. I decided to use a linked list as it was easier to manipulate, add onto and inject seed points into it. There was also a 3D fractal algorithm of this, but I decided that I would make that for another day.

Manga Storm Backup Decoder

Since I use Manga Storm quite a bit, I had been a bit worried about it getting taken down from the app store. So I decided to create a program that will decode their backup which was in quite a simple format (except for a hash function value). I then converted the data that I had into JSON format for me or other people to use in case I wanted to switch apps at some point in time. Still want to add the python arguments module inside as well as some kind of regular expression to decode it far faster and reduce code.

The Real Slime Ritual

The Real Slime Ritual, a Game made in less than 48 hours during Game Jam with some friends. If you really Liked Super Meat boy then your really gonna love this game. The game was created with the open source game engine Superpowers (great for looking up documentation), which was a html5/js designed engine. Best of all it was synchronous, meaning it one person made a mistake then everyone would end up with that mistake at the same time. Pain in the arse when we leave the development server on a locked account (and the person has gone home) at 3am in the morning. Despite that still really fun. Go check it out.

I'll put a link to the game soon. Just have to get the game engine and compile it.
